 Databricks ETL Pipeline Project

## Overview
This project demonstrates an end-to-end ETL pipeline built using Databricks, Apache Spark, and SQL.

## Features
- Data ingestion from CSV files
- Data cleaning and transformation
- Spark DataFrame operations
- SQL-based analytics
- Data export and reporting

## Technologies Used
- Databricks
- Apache Spark
- PySpark
- SQL
- Python

## Project Structure

├── notebooks/

│   └── transformations.py
│
├── datasets/
│   └── sample_data.csv
│
└── README.md

## How to Run

1. Import the notebook into Databricks.
2. Attach a cluster.
3. Run all cells sequentially.
4. Review the output tables and transformations.
